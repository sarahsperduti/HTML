Reproduzindo Áudio

Vamos aprender a reproduzir áudios em HTML5.

Para isso, vamos utilizar a tag <audio>.

Essa tag possui os seguintes atributos:

  • preload: define se o áudio será carregado automaticamente ou não. Os valores possíveis são none, metadata e auto.
  • autoplay: define se o áudio será reproduzido automaticamente ou não. Os valores possíveis são true e false.
  • controls: define se os controles de reprodução serão exibidos ou não. Os valores possíveis são true e false.
  • loop: define se o áudio será reproduzido em loop ou não. Os valores possíveis são true e false.

Além disso, a tag <audio> pode conter uma ou mais tags <source> para especificar os diferentes formatos do arquivo de áudio.

Veja um exemplo de como reproduzir um áudio em HTML5:

<audio preload="metadata" autoplay controls loop>
    <source src="imagens e audio/Apresentação tcc 2024.m4a" type="audio/ogg">
    <p>Infelizmente seu navegador não consegue reproduzir áudio. <a href="midia/guanacast-33.mp3">Clique aqui para baixar o arquivo MP3</a></p>
</audio>
